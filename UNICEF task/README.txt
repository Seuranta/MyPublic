*Учебный проект*

Этот проект - вымышленная задача для проекта UNICEF — международного подразделения ООН, чья миссия состоит в повышении уровня благополучия детей по всему миру. 

Суть проекта — отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, 
находящихся в группе риска.

И сделать это можно с помощью модели, которая предсказывала бы результаты госэкзамена по математике для каждого ученика школы.

Основные этапы:

1. Оценка данных;
2. Обработка данных;
3. Проведение корреляционного анализа;
4. Проведение анализа номинативных переменных
5. Отбор признаков, которые могут использоваться для построения модели.

Информация о данных:

1. Качество данных на среднем уровне, есть пропуски, ошибочные значения и выбросы.
2. Корелляции между столбцами слабые.
