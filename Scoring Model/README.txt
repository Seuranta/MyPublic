*учебный проект*

Цель проекта - построить скоринговую модель, которая по множеству параметров будет относить клиента к одному из классов (выдавать кредит или нет). 

Основные этапы:

1. EDA
2. Обучение модели
3. Построение рекомендательной системы

Информация о данных:

1. Данные разделены на три датасета - тренировочный, тестовый и json-файл
2. В некоторых столбцах было очень много пропусков, поэтому их пришлось сразу удалить
3. Большинство переменных - категориальные
4. Есть много текстовых данных
